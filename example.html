jQuery POP'n SocialButton サンプル

サンプル1: デフォルトのシェアボタン

    $('.social-button.github').popnSocialButton([ 'twitter', 'facebook', 'hatebu', 'gplus' ], {
      url: 'http://github.com/'
    });

サンプル2: GitHubリポジトリのスター数ボタン(誰得)

    $('.social-button.github-repo').popnSocialButton([ 'github' ], {
      githubRepo: 'ktty1220/jquery.popn-socialbutton'
    });

サンプル3: シェア数をふきだし風にカスタマイズ

    $('.social-button.google').popnSocialButton([ 'facebook', 'gplus', 'twitter' ], {
      url: 'http://www.google.com/',
      text: 'ググレカス', 
      imgDir: './img',
      buttonSpace: 108,
      countPosition: {
        top: 8,
        left: 54
      },
      countColor: {
        text: '#222',
        bg: '#eee',
        textHover: '#444',
        bgHover: '#ddd',
        border: '#888'
      },
      countSize: 16
    });
上記に加えて別途CSSを指定
    .social-button.google .popn-socialbutton-count {
      position: relative;
      box-shadow: none !important;
    }
    .social-button.google .popn-socialbutton-count:after,
    .social-button.google .popn-socialbutton-count:before {
      border: solid transparent;
      content: ' ';
      width: 0;
      height: 0;
      left: -16px;
      position: absolute;
    }
    .social-button.google .popn-socialbutton-count:after {
      border-width: 6px;
      border-right-color: #eee;
      top: 5px;
      left: -11px;
    }
    .social-button.google .popn-socialbutton-count:before {
      border-width: 8px;
      border-right-color: #888;
      top: 3px;
    }
    .social-button.google a.popn-socialbutton-count:hover:after {
      border-right-color: #ddd;
    }
